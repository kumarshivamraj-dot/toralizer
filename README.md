# Torializer
**Torializer** is a lightweight, privacy-focused proxy system that routes internet traffic through the [Tor network](https://www.torproject.org/). Designed to be modular and extensible, Torializer aims to make anonymous communication and data transfer easy, programmable, and secure â€” for both developers and end-users.

---

## Features (Completed)
-  **Tor Integration** via SOCKS4
-  **Basic Proxy Logic** to route outgoing requests through Tor
- **C-based Architecture** for performance and speed
- Modular Design: built with future integration in mind
## Goals
- **Integration** of TOR via SOCKS5
- **File-transfer** over tor network
